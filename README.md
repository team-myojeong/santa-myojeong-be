# santa-myojeong-be
## Start

1. 라이브러리 설치

   ```bash
   pip install -r requirements.txt
   ```

2. 마이그레이션

   ```bash
   python manage.py makemigrations
   python manage.py migrate
   ```

3. secrets.json 추가

4. 서버 실행

   ```bash
   python manage.py runserver
   ```



**❗로그인 기능 이용하려면 추가 설정 필요**
